# markdown-web-viewer

https://okaponta.github.io/markdown-web-viewer/

A simple, ephemeral Markdown previewer that runs entirely in your browser. Paste Markdown on the left, see the rendered output on the right in real-time. No data is stored — everything is discarded when you close the tab, making it safe for sensitive content.

Features include a collapsible editor pane, dark mode toggle, and switchable themes (Simple and Kabukicho). Hosted on GitHub Pages with zero build steps — just static HTML, CSS, and JS.
